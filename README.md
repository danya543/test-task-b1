## Документация

Приложение представляет собой платформу для просмотра информации о персонажах из Disney. Адаптив реализован до 400px.
Пользователь может:

- Просматривать карточки персонажей с основной информацией.
- Фильтровать и искать персонажей по фильмам.
- Добавлять персонажей в список избранного.
- Удалять персонажей из избранного.
- Перемещаться по страницам с детальной информацией о персонажах.
- Открывать модальные окна с дополнительной информацией о фильмах.

### Основной функционал

#### 1. Главная страница

Компоненты:

- Mainpage: Основной компонент страницы
- Characters: Отображение карточек персонажей.
  Функционал:
- Загрузка данных о персонажах с помощью fetchCharacters(page, limit).
- Отображение списка фильмов, сериалов и видеоигр.
- Переход к другим персонажам через модальное окно.

#### 2. Страница персонажа

Параметры:

- id: ID персонажа для загрузки данных.

Компоненты:

- CharacterPage: Основной компонент страницы персонажа.
- CharacterInfo: Выводит основную информацию о персонаже (имя, изображение, фильмы, сериалы, видеоигры).
- ExpandableBlock: Раскрывающийся список для длинных текстов (список сериалов или видеоигр).
- ModalPortal: Модальное окно для отображения информации о фильмах.
- Фильтрация и поиск по фильму.
- OtherCharacters: Список других персонажей, участвующих в выбранном фильме.

Функционал:

- Загрузка данных о персонаже с помощью fetchCharacter(id).
- Отображение списка фильмов, сериалов и видеоигр с участием персонажа.
- По клику на фильм отображение списка персонажей участвующих в нем.
- Переход к другим персонажам через модальное окно.

#### 3. Список персонажей

Компоненты:

- Card: Карточка персонажа с изображением, именем, фильмами и видеоиграми.
- Bookmark: Кнопка добавления/удаления из избранного.
- Фильтрация количества персонажей на странице.
- Пагинация по страницам.
- Релизован skeleton loader отображаемый при загрузке контента.

Функционал:

- Переход на страницу персонажа при клике на карточку.

#### 4. Избранное

Функционал:

- Загрузка списка ID избранных персонажей из local storage.
- Хранение списка ID избранных персонажей в Redux.
- Возможность добавления/удаления персонажей в избранное.
- Отображение избранных персонажей.
- Реализован свайпер при большом количестве сохраненных персонажей.
- Удаление списка избранных с подтверждением через модальное окно Confirm.

#### 5. Утилиты

Компоненты:

- Alert: компонент для отображения уведомления для пользователя.
- Button: ui для кнопки.
- Select: компонент для фильтрации количества персонажей на странице.

### Хуки

#### 1. useBurgerMenu

Хук для управления бургер-меню.

Функционал:

- Функции для открытия/закрытия меню.
- Отображение иконки меню.

#### 2. useClickOutside

Хук для закрытия модальных окон при клике вне их блока.

### Стили

В файле theme.scss собраны все scss переменные используемые для приложения.

### Основные модули

#### 1. API

**fetchCharacter(id)**

- Загружает информацию о конкретном персонаже.
- Возвращает объект Character.

**fetchInfo({ name, film, page })**

- Загружает список персонажей по фильму или имени, с заданной страницей.
- Возвращает объект с данными и метаинформацией (CharactersResponse).

#### 2. Redux

**favoriteSlice**

state.favorite.items: Хранит массив ID избранных персонажей.

Экшены:

- addFavorite(id): Добавляет персонажа в избранное.
- removeFavorite(id): Удаляет персонажа из избранного.
- clearFavorites(): Полностью очищает список избранного.

**characterSlice**

state.characters.characters: Хранит массив данных о персонажах.
state.characters.info: Хранит информацию для пагинации персонажей.
state.characters.currentPage: Хранит текущую страницу.
state.characters.perPage: Хранит количество персонажей на страницу.
state.characters.status: Хранит статус запроса данных с API.
state.characters.error: Хранит ошибку.

Функционал:

- Асинхронный запрос данных с API
- Обработка запроса и переопределение статуса и соответствующих переменных.

### Основные компоненты

#### 1. Card

Карточка персонажа с информацией и функционалом добавления в избранное.

**Пропсы:**

- name, imageUrl, \_id, films, videoGames — данные персонажа.

**Особенности:**
Кнопка "Избранное" обрабатывается через компонент Bookmark.
Клик по карточке переходит на страницу персонажа.

#### 2. BurgerMenu, Burger

Компоненты для отображения бургер-меню

**Пропсы:**

- icon: иконка меню(Burger).
- handleToggle: изменение состояния отслеживающего открыто меню или закрыто(Burger).
- isOpen: состояние меню откыто/закрыто(BurgerMenu).
- handleClose: функция для закрытия меню(BurgerMenu).

#### 3. ModalPortal

Универсальное модальное окно для отображения контента.

**Пропсы:**

- onClose: Закрытие модального окна.
- children: Контент, отображаемый внутри модального окна.

#### 4. OtherCharacters

Компонент для отображения других персонажей, связанных с фильмом.

**Пропсы:**

- name: Имя персонажа (опционально).
- film: Название фильма (опционально).
- onClose: Закрытие модального окна.

**Особенности:**
Пагинация для большого количества персонажей.

#### 5. Confirm

Модальное окно для подтверждения действий.

**Пропсы:**

- confirmText: Текст подтверждения.
- onClose: Закрытие окна.
